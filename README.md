# Noah Lach — Soccer Recruiting Website

Personal recruiting site for Noah Lach, Class of 2028 — MLS Next Academy Division (Sporting City), Kansas City, MO.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Player profile, stats, highlights, academics, contact |
| `colleges.html` | In-depth analysis of 30 target schools (division, academic fit, feasibility, coach contacts) |
| `map.html` | Interactive map of all target colleges with filters and region fly-to |
| `scholarships.html` | Athletic scholarship opportunities and research |
| `camps.html` | ID camp schedule and target events |
| `testprep.html` | ACT/SAT prep plan and test timeline |

## Stack

Static HTML/CSS/JS — no frameworks, no build step. Hosted via GitHub Pages.

- Fonts: Google Fonts (Bebas Neue, Barlow Condensed, DM Sans)
- Map: [Leaflet.js](https://leafletjs.com/) + CartoDB Voyager tiles
